## Freefish 咸鱼社区

### 在线体验

http://freefish.love

### 项目主要业务及实现的功能

本项目类似一个简版的掘金这样的技术社区，实现了多个用户注册，登录，发帖，评论，关注，搜索等功能。

### 前端技术栈

Axios

Bulma

Element

### 后端技术栈
